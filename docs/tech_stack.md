# Technology Stack

The `website_anti` portfolio is built using modern web standards to ensure speed, accessibility, and a premium visual experience.

## Front-end Core

- **HTML5**: Semantic structure for better SEO and accessibility.
- **Vanilla CSS3**: Custom styles without the overhead of heavy frameworks, utilizing modern features like CSS variables and Flexbox/Grid.
- **Vanilla JavaScript (ES6+)**: Custom animations and interactive elements.

## Libraries & Integration

- **Particles.js**: For the dynamic, interactive particle background in the hero section.
- **Lucide Icons**: Clean and consistent iconography throughout the site.
- **Google Fonts**: Using 'Inter' for readability and 'Outfit' for a premium, modern feel.

## Documentation Engine

- **MkDocs**: Static site generator geared towards project documentation.
- **Material for MkDocs**: A powerful theme providing a professional, responsive interface with search and navigation capabilities.
